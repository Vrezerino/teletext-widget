# Teksti-TV-työpöytä-widget

<video controls src="https://github.com/Vrezerino/teksti-tv-widget/raw/refs/heads/main/ttw-kaytto.mp4" title="Title"></video>

Tämä ohjelma tarjoaa pienessä ikkunassa työpöydällesi Yle:n teksti-tv:n sekä perinteiset kuvamuotoiset että tekstimuotoiset sivut.

Ohjelmassa voi lisätä ja poistaa kirjanmerkkejä samalla painikkeella; se poistaa sivun kirjanmerkeistä jos nykyinen sivu-alasivu-kombinaatio on jo kirjanmerkeissä. Kirjanmerkki lisätään listan loppuun, mutta niitä voi poistaa mistä kohdasta tahansa.

Ohjelmaikkunaa voi sävyttää eri väreillä tai sen voi muuttaa mustavalkoiseksi. Sivujen tyyliä voi vaihtaa niin ikään yhdellä napilla.

Toistaiseksi ohjelma on saatavilla Windowsille. Macintosh- ja Linux-buildit vaativat enemmän testaamista ennen kuin laitan ne jakoon.

This is a teletext desktop widget program that gets the teletext pages from Finnish Broadcasting Company Yle. One can save pages as bookmarks, adjust program window hue, set it as black-and-white and toggle the theme between traditional and "modern" looks.
